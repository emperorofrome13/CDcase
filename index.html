Spotify setup & import

To log in and pull your saved albums, this file must be served from a loopback address (Spotify no longer allows localhost or plain web http). One-time setup:

  1. Save this file as index.html in a folder, then in that folder run: python3 -m http.server 8888 --bind 127.0.0.1
  2. Open http://127.0.0.1:8888/ in your browser.
  3. At developer.spotify.com/dashboard create an app (Web API), and add this exact Redirect URI:
    —
  4. Copy the app's Client ID and paste it here:
  5. Click Connect Spotify up top, approve, then use Import saved albums.

No login needed for the rest of the app — pasting links, pinning, playing and printing all work on their own.
